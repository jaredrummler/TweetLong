# TweetLong

tweetlong - A command-line tool to split tweets, built in Kotlin

## Build

```gradle
./gradlew :runReleaseExecutableMacos
```

Copy executable in build/bin/macos to /usr/bin/local/tweetlong

## Usage:

```sh
$ tweetlong --tweet "REPLACE WITH TWEET"
```
